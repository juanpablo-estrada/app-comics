# app-comics

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

### Español
Cómo correr el proyecto:

1. Clonar el repositorio.
2. npm install.
3. npm run dev.

En la página principal se abrirá un modal que pedirá un nombre. Esto es para tener más interacción con el usuario. Cuando se cierre el modal, se mostrará un cómic principal que se generará aleatoriamente. Debajo de la imagen del cómic encontrarás 3 botones que son:

- El PREV, que es para ver el cómic anterior al que está actualmente.
- El RANDOM, que generará un cómic aleatorio.
- El NEXT, que mostrará el cómic siguiente al actual.

También hay un sistema de calificación de cómics. Cuando califiques un cómic, el sistema guardará tu calificación siempre.

### Ingles

How to run the project:

1. Clone the repository.
2. npm install.
3. npm run dev.

On the main page, a modal will open asking for a name. This is for more interaction with the user. When the modal is closed, a primary comic will be displayed, generated randomly. Below the comic image, you will find 3 buttons:

- PREV, which is for viewing the previous comic from the current one.
- RANDOM, which generates a random comic.
- NEXT, which will display the next comic after the current one.

There is also a comic rating system. When you rate a comic, the system will always save your rating.
```
